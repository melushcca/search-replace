# search-replace



- Replace Multiple Values at once
- Replace Strings in Array with other Strings in other Array and get the output
- Replaces also:
  - u00BD (½)
  - u00BE (¾)
  - u00B0 (°)
  - u00BC (¼)
  - u0040 (@)
  - u0021 (!)
  - u0025 (%)
  - u002A (*)
  - u0028 (()
  - u0029 ())
  - u00B2 (²)
  - u003D (=)
  - u00AE (®)
